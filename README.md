# TuneWise AI 🎵

Асинхронное backend-приложение для управления музыкальной медиатекой. Проект демонстрирует интеграцию современного веб-фреймворка, векторной базы данных для ИИ-поиска, фонового воркера для тяжелых задач и интеграции с внешними API языковых моделей.

---

## 🚀 Основной функционал

1. **REST API:** Полностью асинхронный API на FastAPI с валидацией данных через Pydantic v2.
2. **Семантический ИИ-поиск (pgvector):** Поиск треков не по ключевым словам, а по смыслу и настроению текстового запроса (например, *"грустная акустика для дождливой осени"*).
3. **Фоновые задачи (Celery + RabbitMQ + Redis):** При создании нового плейлиста задача анализа треков и генерации креативного обзора улетает в фоновый воркер Celery, сохраняя отзывчивость основного API.
4. **Интеграция с ИИ (OpenRouter API):** Генерация саммари плейлистов с помощью бесплатных LLM моделей (через единый интерфейс OpenRouter).
5. **Отказоустойчивость (Fallback Mode):** В случае отсутствия платного баланса на OpenRouter для эмбеддингов, система автоматически переключается на локальный детерминированный алгоритм генерации векторов, не ломая работу приложения.

---

## 🛠 Стек технологий

* **Язык:** Python 3.11+
* **Фреймворк:** FastAPI (Async/Await)
* **ORM:** SQLAlchemy 2.0 (асинхронный режим)
* **База данных:** PostgreSQL 16 + расширение **pgvector**
* **Очередь задач:** Celery
* **Брокер сообщений:** RabbitMQ
* **Бэкенд результатов Celery:** Redis
* **Валидация:** Pydantic v2
* **ИИ-интеграция:** OpenRouter API (OpenAI Compatible SDK)
* **Окружение:** Docker & Docker Compose
